# BOSS直聘候选人智能筛选助手 — 使用指南

> 版本: v0.1.0 | 更新日期: 2026-03-16 | 适用角色: 招聘运营、开发、产品

---

## 这是什么

一个浏览器插件脚本（油猴脚本），安装到 BOSS直聘 网页后，会**自动对推荐的候选人打分并高亮**，帮你在海量简历中快速识别匹配的人。

**核心价值**: 把人工扫简历的 5 秒/人 压缩到 0.5 秒/人。

---

## 效果一览

| 颜色 | 含义 | 你该怎么做 |
|------|------|-----------|
| 绿色边框 + 绿色徽章 | 高匹配（分数 >= 70） | 优先沟通 |
| 黄色边框 + 橙色徽章 | 一般匹配（分数 40-69） | 酌情考虑 |
| 半透明 + 灰色徽章 | 不匹配（分数 < 40） | 跳过 |

鼠标悬停分数徽章可看到**评分明细**（哪些规则加了分、扣了分）。

高分候选人出现时，右上角会弹出 Toast 通知，可一键定位到该卡片。

## 安装步骤（5 分钟）

### 第一步：安装 Tampermonkey

| 浏览器 | 安装地址 |
|--------|---------|
| Chrome | Chrome 应用商店搜索 "Tampermonkey" |
| Edge | Edge 扩展商店搜索 "Tampermonkey" |
| Firefox | Firefox 附加组件搜索 "Tampermonkey" |

安装后浏览器右上角会出现一个黑色方块图标。

### 第二步：安装脚本

1. 点击 Tampermonkey 图标 → "添加新脚本"
2. 删除编辑器里的所有内容
3. 把 `boss-helper.user.js` 的全部代码粘贴进去
4. 按 `Ctrl+S` 保存
5. 关闭编辑器标签页

### 第三步：选择你的角色配置

1. 打开 [BOSS直聘](https://www.zhipin.com/) 并登录
2. 页面右下角会出现一个蓝色 **齿轮按钮** ⚙ → 点击它
3. 面板顶部「配置源」→ 下拉选择你的岗位角色
4. 点击「立即同步」
5. 看到"同步成功"后关闭面板，完成

| 角色 | 说明 | 适用人群 |
|------|------|---------|
| 开发岗 | 通用技术岗位筛选规则 | 招前端/后端/全栈 |
| 算法岗(C++) | C++算法工程师专用，含金融背景加分 | 招量化/算法 |
| 运营岗 | 弱化学历权重，强调运营经验关键词 | 招运营 |
| 产品岗 | 弱化技术关键词，强调产品思维 | 招产品经理 |

> 配置会自动从 GitHub 同步。管理员更新了规则后，你下次刷新页面就会自动生效，**无需任何操作**。

## 日常使用

安装完成后，**无需任何额外操作**。正常浏览 BOSS直聘 推荐候选人页面即可，脚本会自动：

1. 拦截候选人数据
2. 按配置规则逐条评分
3. 给每张卡片打上分数徽章和颜色高亮
4. 高分候选人弹出通知提醒

### 通知行为

- 候选人分数达到设定阈值（默认 70 分）时触发通知
- 通知方式：页面右上角 Toast + 桌面系统通知 + 提示音
- 在配置面板「高级设置」中可调整通知分数下限，或关闭通知/提示音

---

## 配置面板说明

点击右下角齿轮 ⚙ 打开面板，从上到下依次是：

| 区块 | 作用 | 是否需要改 |
|------|------|-----------|
| 配置源 | 选择角色、同步远程配置 | 首次选好后一般不动 |
| 预设模板 | 一键切换本地预设（技术岗通用/严格/宽松） | 可选，与远程配置独立 |
| 评分阈值 | 调整绿/黄/灰的分数门槛 | 按需调 |
| 评分规则 | 查看/编辑/增删评分规则 | 通常不需要改 |
| 加分关键词 | 简历中出现这些词会加分 | 按需增减 |
| 减分关键词 | 简历中出现这些词会扣分 | 按需增减 |
| 院校名单 | 内置 985/211 名单，可添加自定义院校 | 通常不改 |
| 高级设置 | 通知开关、通知分数下限、探测模式、调试模式 | 按需 |

底部三个按钮：

- **导出配置** — 下载当前配置为 JSON 文件（备份用）
- **导入配置** — 从 JSON 文件恢复配置
- **恢复默认** — 重置为出厂设置

## 评分规则简述

以「算法岗(C++)」配置为例：

### 加分项

| 规则 | 条件 | 分值 |
|------|------|------|
| 硕士及以上 | 学历为硕士/博士/MBA | +15 |
| 985 院校 | 毕业于 39 所 985 高校 | +25 |
| 211 院校 | 毕业于 211 高校 | +15 |
| 海外名校 | MIT/Stanford/清华等 | +20 |
| 1-7年经验 | 工作年限 1-7 年 | +15 |
| 大厂经历 | 任职过 BAT/华为/美团等 | +15 |
| ACM竞赛获奖 | 简历含 ACM/ICPC/NOI/IOI | 每命中 +20 |
| 金融背景 | 量化/券商/期货/交易系统等 | 每命中 +10 |
| 加分关键词 | C++/算法/多线程/CUDA等 | 每命中 +10 |

### 扣分项

| 规则 | 条件 | 分值 |
|------|------|------|
| 大专及以下 | 学历为大专/高中/中专/初中 | -50 |
| 低层次院校 | 校名含"职业技术学院"等 | 每命中 -20 |
| 非全日制 | 自考/函授/成人教育/电大等 | 每命中 -30 |
| 非C++职位 | "Java开发""Python工程师"等 | 每命中 -20 |
| 非计算机专业 | 土木/农学/文学/医学等 | 每命中 -40 |
| 恒生万德 | 简历含恒生/万得/Wind | 每命中 -15 |
| 减分关键词 | 外包/培训/实习/兼职 | 每命中 -15 |

> 所有候选人的基准分为 50 分，加减分后限制在 0-100 之间。

## 常见问题

### Q: 安装后页面没有任何变化？

A: 脚本依赖 BOSS直聘 的 API 数据。请确保你在**推荐牛人**或**搜索候选人**页面，而非首页。首次加载可能需要等待 1-2 秒。

### Q: 评分不准怎么办？

A: 评分基于关键词匹配和规则引擎，不可能 100% 准确。可以在配置面板微调关键词和规则分值。调整后即时生效。

### Q: 换了岗位招聘，怎么切换规则？

A: 齿轮 ⚙ → 配置源 → 下拉切换角色 → 立即同步。

### Q: 断网了还能用吗？

A: 能。远程配置会缓存在本地，断网时使用上次同步的配置。

### Q: 会影响 BOSS直聘 正常使用吗？

A: 不会。脚本只读取页面数据进行分析，不会修改任何数据、不会自动发消息、不会触发任何操作。
